前言

JDBC代表Java数据库连接。JDBC库中所包含的API通常与数据库使用于：连接到数据库创建SQL或MySQL语句，在数据库中执行SQL或MySQL，查看和修改数据库中的数据记录。

一、JDBC环境准备

  • 普通Java项目，在lib中粘贴此包（在官网上下），版本无所谓

官网下载 系统选择platform那个

jar包存放位置，项目根目录下的lib文件夹

  • 如果是Maven或者Gradle或者Spring项目。不需要引入上面的jar包，直接在依赖里面写（Maven为例）
